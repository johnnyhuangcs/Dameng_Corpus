

# 数据库目录结构介绍

## 一、数据库安装目录

下图展示为 DM8 数据库目录。

![image.png](https://eco.dameng.com/eco-file-server/file/eco/preview/20240130111649EE9TU61KD3BOUK6YON)

/home/dmdba/dmdbms/bin 目录存放 DM 数据库的可执行文件，例如 disql 命令、dminit 命令、dmrman 工具等。

![image.png](https://eco.dameng.com/eco-file-server/file/eco/preview/20240130111852XJRP76WJCWROUGPCKO)

![image.png](https://eco.dameng.com/eco-file-server/file/eco/preview/20240130112133AN0Q8GUUTZCO6BR5N5)

/home/dmdba/dmdbms/drivers 存放连接 DM 数据库的驱动文件。

![image.png](https://eco.dameng.com/eco-file-server/file/eco/preview/20240130112242AIFMLHHNUY7CIHYZ8L)

/home/dmdba/dmdbms/log 存放 DM 数据库日志，包括工具的日志、数据库日志、服务日志等。

![image.png](https://eco.dameng.com/eco-file-server/file/eco/preview/2024013011235450OT8JW6JB9H5CD6SM)

/home/dmdba/dmdbms/samples 存放 DM 数据库各类配置文件的示例文件。

/home/dmdba/dmdbms/script/root 存放注册、注销 DM 数据库服务的工具，例如 dm_service_installer.sh 等。

![image.png](https://eco.dameng.com/eco-file-server/file/eco/preview/202401301126108BS3DLCU0XUREQLMCQ)

/home/dmdba/dmdbms/tool 存放 DM 数据库的各个工具，例如 manager 管理工具、dbca 数据库配置助手等。

## 二、数据库实例目录

/dmdata/data 为数据库实例目录，该目录存放各个实例的文件。

![image.png](https://eco.dameng.com/eco-file-server/file/eco/preview/20240130113403A8653ELPKCIZTZLJR6)

以实例 DAMENG 为例，该目录下存放 DAMENG 实例的配置文件 `*.ini、控制文件 dm.ctl、数据文件 *.DBF、日志文件 *.log` 等。

![image.png](https://eco.dameng.com/eco-file-server/file/eco/preview/20240130114133H84932DACAEGUA0VNG)
